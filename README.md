### Prerequisites

- [Node.js](https://nodejs.org/)

### Installation

1. Clone the repository

```bash
git clone https://github.com/RoniGuru/ProjectManagerElectron.git
cd ProjectManagerElectron
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
